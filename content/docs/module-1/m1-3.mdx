# Data structures

## What are data structures

The data structures of JavaScript are:

- Array
- Object
- Set
- Map

## Why do we need data structures

Different tasks need to have different ways of accessing data, and for doing that we got different types of data structures.
There are certain tasks that we need to perform but it's not possible with objects, but is possible with arrays, and vice versa.
