## Naming

Use meaningful names for variables. Naming cases:

Valid:

- camel
- snake

Invalid:

- kebab

## What a data type does

A data type tells Go:

- What kind of data a variable holds
- How much memory to allocate
- What operations are allowed

## Data types in Go

### Integer

#### int (integer)

- int
- int8
- int16
- int32
- int64

Just use `int` if you're not sure which one to use, Go does the memory allocation behind the scenes.

#### uint (unsigned integer)

`uint` is the same as int, but it only stores positive numbers.

### Float

- float
- float32
- float64

This one can also be simply typed as `float`, go does the things in the background.

### Boolean

- bool

Boolean (`bool`) stores `true` and `false`.

### String

- string

String stores text data in quotations.
