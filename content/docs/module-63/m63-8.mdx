Example code for function:

```go
package main

import "fmt"

func coffee(coffeeType string) {
	fmt.Printf("Making %v Coffee...\n", coffeeType)
}

func main() {
	coffee("Africano")
}
```

This function receives a string type parameter `coffeeType`.

Multiple parameters can be received by just using a command and adding another. Just like in TypeScript.
