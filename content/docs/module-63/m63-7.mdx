## Zero Values

Zero values are the values a data type initially has, before we assign something oursleves.

Default values for different data types:

| type   | value |
| ------ | ----- |
| int    | 0     |
| string | ""    |
| float  | 0.0   |
| bool   | false |

## Print Commands

| Code     | Definition                                                       |
| -------- | ---------------------------------------------------------------- |
| Print    | prints                                                           |
| Println  | prints, adds a `\n` in the end, adds spaces between values       |
| Printf   | prints, enables using Print Verbs                                |
| Scan     | Takes user input                                                 |
| Sprint   | Works same as Print. Returns the output instead of printing it   |
| Sprintln | Works same as Println. Returns the output instead of printing it |
| Sprintf  | Works same as Printf. Returns the output instead of printing it  |

## Commands From Other Packages

| Package   | Code       | Definition                                                                                |
| --------- | ---------- | ----------------------------------------------------------------------------------------- |
| math/rand | Intn()     | Generates a random number between 0 to its parameter                                      |
| math/rand | Seed()     | Changes the random number generation seed so that the random number isn’t same every time |
| time      | UnixNano() | Returns the nanosecond of the parent time function. Mostly used with Now()                |
