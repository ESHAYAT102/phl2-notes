## Getting Things Ready

Create a project directory for Go (mkdir code/go)
Create a `main.go` file in the project directory (touch code/go/main.go)

## Write the code

Simple "Hello World":

```go title="main.go"
package main

import (
    "fmt"
)

func main() {
    fmt.Println("Hello World!")
}
```

## Run The Code

Enter `go run main.go` in the terminal to run the code. Make sure you're in the right directory.
