---
title: Module 22.10
description: Upsert in prisma (update and delete)
---

Use `upsert` to create a data if it doesn't exist or update it if it already exists.
Takes the `update` key to receive the data that needs to be updated, and the `create` key takes in the keys that needs to be created if teh data doesn't exist. Must use the `where` key to find the dedicated data.
