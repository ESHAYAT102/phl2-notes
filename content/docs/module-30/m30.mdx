হ্যালো ডেভেলপারস !!

ইয়েস! চলে আসলো আরেকটা এক্সাইটিং মডিউল। এতদিন Prisma, Express ও PostgreSQL ব্যবহার করে একটি Blog Application এর Backend তৈরি করেছেন। যেখানে POST, GET, UPDATE & DELETE মেথড ব্যবহার করে প্রয়োজনীয় সব API বানিয়েছি।

এবার সময় এসেছে সেই ব্যাকএন্ডকেে ফ্রন্টএন্ডে ব্যবহার করার। এখন আমরা দেখব কীভাবে আমাদের তৈরি করা Blog Backend API কে একটি Next.js ব্যবহার করে ফ্রন্টএন্ডের সাথে ইমপ্লিমেন্ট করা যায়।
