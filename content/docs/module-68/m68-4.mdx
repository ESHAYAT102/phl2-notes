description: Defer Basics
---

Use the `defer` keyword before any line to make them execute in the end of the function.
It's like a timeout.

Example:

```go
package main

import "fmt"

func deferred(result int) {
	fmt.Println("Deferred result:", result)
}

func output() int {
	number := 16
	defer deferred(number)
	fmt.Println("From `output` function:", number)
	return number
}

func main() {
	output()
}
```

Output:

```bash
From `output` function: 16
Deferred result: 16
```

---

Defer will values like this:

```go
package main

import "fmt"

func deferred(result int) {
	fmt.Println("Deferred result:", result)
}

func output() int {
	number := 16
	defer deferred(number)
	fmt.Println("From `output` function:", number*2) // [!code highlight]
	return number
}

func main() {
	output()
}
```

Output:

```bash
From `output` function: 32
Deferred result: 16
```
