---
title: Module 6.3
description: Generics in typescript
---

```ts
type GenericArray<value> = Array<value>;

const str: GenericArray<string> = ["apple", "mango", "banana"];

const int: GenericArray<number> = [3, 5, 2];

const bool: GenericArray<boolean> = [true, true, false];
```

Using `GenericArray<type>` instead of `Array<type>`.

A better example:

```ts
type Coordinates<X, Y> = [X, Y];

const coordinates: Coordinates<number, number> = [12, 32];
```
